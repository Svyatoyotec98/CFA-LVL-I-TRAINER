## ОСНОВНЫЕ WORKSHEETS

### 1. TVM (Time Value of Money)
**Доступ:** Прямые клавиши N, I/Y, PV, PMT, FV

| Клавиша | Переменная | Описание |
|---------|------------|----------|
| N | N | Количество периодов |
| I/Y | I/Y | Процентная ставка за год |
| PV | PV | Текущая стоимость |
| PMT | PMT | Периодический платёж |
| FV | FV | Будущая стоимость |

**Подготовка:**
```
[2ND] [CLR TVM]     — очистить TVM
[2ND] [P/Y] n [ENTER] — установить периодов в год
[2ND] [BGN/END]     — начало/конец периода
```

**Типичный расчёт FV:**
```
[2ND] [CLR TVM]
[2ND] [P/Y] 4 [ENTER]  — квартальное начисление
[2ND] [QUIT]
150000 [+/-] [PV]
8 [I/Y]
3 [2ND] [xP/Y] [N]     — или 12 [N]
0 [PMT]
[CPT] [FV]
```

### 2. Cash Flow Worksheet
**Доступ:** [CF]

| Клавиша | Переменная | Описание |
|---------|------------|----------|
| CF | CF0 | Начальный денежный поток |
| ↓ | C01, C02... | Денежные потоки периодов |
| ↓ | F01, F02... | Частота повторения потока |
| IRR | IRR | Внутренняя норма доходности |
| NPV | NPV | Чистая текущая стоимость |

**Расчёт IRR (MWRR):**
```
[CF]
[2ND] [CLR WORK]
30 [+/-] [ENTER] [↓]    — CF0 = -30
42 [+/-] [ENTER] [↓]    — C01 = -42
1 [ENTER] [↓]           — F01 = 1
110 [ENTER] [↓]         — C02 = 110
1 [ENTER]               — F02 = 1
[IRR] [CPT]             — Результат: 33.88%
```

**Расчёт NPV:**
```
[CF]
... (ввести потоки)
[NPV]
10 [ENTER]              — I = 10%
[↓] [CPT]               — Результат: NPV
```

### 3. Statistics Worksheet
**Доступ:** [2ND] [DATA]

**Ввод данных:**
```
[2ND] [DATA]
[2ND] [CLR WORK]
X01: 15 [ENTER] [↓]
Y01: 1 [ENTER] [↓]      — частота (если нужна)
X02: 10 [ENTER] [↓]
...
[2ND] [STAT]
[↓] до n, x̄, Sx, σx
```

**Переменные:**
| Дисплей | Описание |
|---------|----------|
| n | Количество наблюдений |
| x̄ | Среднее арифметическое |
| Sx | Стандартное отклонение (выборка) |
| σx | Стандартное отклонение (популяция) |

---

### 4. Bond Worksheet
**Доступ:** [2ND] [BOND]

| Переменная | Описание |
|------------|----------|
| SDT | Settlement date |
| CPN | Coupon rate (annual %) |
| RDT | Redemption date |
| RV | Redemption value (100 = par) |
| ACT/360 | Day count convention |
| 2/Y, 1/Y | Купонов в год |
| YLD | Yield to maturity |
| PRI | Price |

---

### 5. Interest Conversion
**Доступ:** [2ND] [ICONV]

| Переменная | Описание |
|------------|----------|
| NOM | Nominal rate |
| EFF | Effective rate |
| C/Y | Compounding per year |

**Расчёт EAR:**
```
[2ND] [ICONV]
8 [ENTER]               — NOM = 8%
[↓] 4 [ENTER]           — C/Y = 4
[↓] [CPT]               — EFF = 8.24%
```

## ЧАСТЫЕ ЗАДАЧИ CFA

### 1. Future Value с компаундингом
```
Дано: PV=150,000, r=8%, quarterly, 3 года
[2ND] [CLR TVM]
[2ND] [P/Y] 4 [ENTER] [2ND] [QUIT]
150000 [+/-] [PV]
8 [I/Y]
12 [N]          — или 3 [2ND] [xP/Y] [N]
0 [PMT]
[CPT] [FV]      → 190,236.27
```

### 2. Present Value аннуитета
```
Дано: PMT=1000, r=6%, n=10
[2ND] [CLR TVM]
1000 [+/-] [PMT]
6 [I/Y]
10 [N]
0 [FV]
[CPT] [PV]      → 7,360.09
```

### 3. IRR (Money-weighted return)
```
Дано: CF0=-1000, CF1=-500, CF2=2000
[CF] [2ND] [CLR WORK]
1000 [+/-] [ENTER] [↓]
500 [+/-] [ENTER] [↓] 1 [ENTER] [↓]
2000 [ENTER] [↓] 1 [ENTER]
[IRR] [CPT]     → IRR%
```

### 4. Geometric Mean
```
Дано: Returns 15%, 10%, 12%, 3%
1.15 [×] 1.10 [×] 1.12 [×] 1.03 [=]
[yˣ] 0.25 [=]
[-] 1 [=]       → 0.099 = 9.9%
```

### 5. Continuously Compounded Return
```
Дано: HPR = 1.2%
1.012 [LN]      → 0.01192 = 1.19%
```

### 6. EAR из Nominal
```
Дано: Nominal 8%, quarterly
[2ND] [ICONV]
8 [ENTER] [↓]
4 [ENTER] [↓]
[CPT]           → EFF = 8.24%
```
